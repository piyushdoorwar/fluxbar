Small Surface, Real Control

The settings window is the feature list.

Live speed at a glance

Show separate upload and download values, or collapse them into one total speed indicator.

Compact formats

Choose standard text or compact labels like 120K / 35K and 120K↓ 35K↑.

Network source choice

Use automatic detection, Wi-Fi, Ethernet, or all recognized real interfaces.

Usage history

Review daily download, upload, and total usage from the last 30 days.

Bytes or bits

Switch speed units depending on whether you prefer file-transfer style or network-rate style numbers.

Idle-aware label

Hide the top bar indicator when the selected network source has no active traffic.

Hover breakdown

Hover the panel label to see download, upload, and total speed details in one compact tooltip.

Panel styling

Use normal or bold text and optionally set a custom top bar label color.

Refresh choices

Pick a 1, 2, 3, or 5 second update interval to balance freshness with calmness.

Local counters

Reads Linux interface counters from /proc/net/dev and ignores virtual adapters in automatic mode.

No telemetry

No accounts, analytics, network calls, or external services. Usage history stays on your machine.

Interactive Preview

Tune the label before you install.

Install Locally

Copy it into your GNOME extensions folder.

FluxBar supports GNOME Shell 45 through 50 and can be installed directly from this repository.

mkdir -p ~/.local/share/gnome-shell/extensions
rsync -a --delete --exclude='.git' ./ ~/.local/share/gnome-shell/extensions/fluxbar@piyushdoorwar.github.io/
glib-compile-schemas ~/.local/share/gnome-shell/extensions/fluxbar@piyushdoorwar.github.io/schemas
gnome-extensions enable fluxbar@piyushdoorwar.github.io
